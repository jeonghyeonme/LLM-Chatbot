# 🎭 Dual-Persona Counseling Chatbot

사용자의 고민에 대해 두 개의 서로 다른 페르소나가 동시에 답변하고, 서로 상호작용하며 최선의 해결책을 찾아가는 지능형 상담 챗봇 프로젝트입니다.

## 🌟 핵심 컨셉: "이성과 감성의 조화"
단순히 질문에 답하는 것을 넘어, 성격이 다른 두 페르소나가 사용자의 고민을 두고 토론하거나 보완적인 조언을 제공함으로써 사용자에게 다각적인 통찰을 제공합니다.

### 👥 주요 페르소나 및 역할
본 시스템은 두 명의 상반된 전문가가 한 팀이 되어 상담을 진행하는 경험을 제공합니다.

1. **이성적 분석가**
   - **특징**: 논리적, 객관적, 현실 중심적.
   - **역할**: 문제의 원인을 냉철하게 분석하고, 실행 가능한 단계별 솔루션을 제안합니다.
   - **말투**: 명확하고 간결한 "~입니다" 체.

2. **감성적 공감자**
   - **특징**: 공감적, 따뜻함, 감정 중심적.
   - **역할**: 사용자의 상처받은 마음을 어루만지고, 정서적 지지와 심리적 안정감을 제공합니다.
   - **말투**: 부드럽고 다정한 "~예요" 체.

### 🔄 상호작용 메커니즘 (Persona Interaction)
두 페르소나는 단순 병렬 답변에 그치지 않고 **Function Calling** 기술을 통해 서로 대화합니다.
- **상호 보완**: 엘리아의 현실적인 조언에 루나가 정서적 배려를 더함.
- **건설적 토론**: 해결 방식에 이견이 있을 경우, 시스템 내부에서 두 페르소나가 짧은 토론을 거쳐 사용자에게 최선의 절충안을 제시.

## 🛠️ 기술 구현 전략 (LLM 교안 활용)
프로젝트는 검증된 LLM 기술들을 조합하여 견고한 상담 시스템을 구축합니다.

- **Orchestration**: `gpt-4o` 모델을 기반으로 오케스트레이터를 구축하여 대화 흐름을 제어합니다.
- **Function Calling (07주차)**: 페르소나 간의 상호작용(토론, 동의, 보완)을 유발하는 트리거로 활용합니다.
- **Parallel Processing**: 두 페르소나의 개별 답변 생성을 병렬화하여 응답 지연을 최소화합니다.
- **Whisper API (05주차)**: 음성 인식 기술을 도입하여 사용자가 말로 직접 고민을 털어놓을 수 있는 환경을 제공합니다.
