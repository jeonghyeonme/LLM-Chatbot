# 📢 기획 발표 개요 (Presentation Outline)

발표 시 핵심적으로 전달해야 할 메시지와 구성안입니다.

## 1. 문제 제기 (Background)
- 기존 챗봇의 한계: 단편적이고 일관된 하나의 답변만 제공하여, 사용자의 복합적인 감정이나 상황을 충분히 반영하지 못함.
- 필요성: 고민 상담은 정답이 없으며, 다양한 관점(이성적 조언 + 정서적 위로)의 공존이 필요함.

## 2. 해결책: 듀얼 페르소나 상호작용 (The Solution)
- **Concept**: 한 번의 질문에 두 개의 심장이 답하다.
- **Key Feature**: 두 페르소나가 서로의 의견에 피드백을 주고받는 '상호작용'을 통해 상담의 깊이를 더함.

## 3. 핵심 기술 (Core Tech)
- **GPT-4o + Parallel Calling**: 동시 답변 생성.
- **Agent Orchestration via Function Calling**: 페르소나 간 토론 메커니즘.
- **Multi-modal Interface**: 음성(Whisper)을 통한 실감 나는 상담 경험.

## 4. 기대 효과 (Impact)
- 사용자 맞춤형 다각도 조언 제공.
- 챗봇과의 대화가 아닌, '상담 팀'과 대화하는 듯한 풍부한 사용자 경험.
- 감정적 해소와 현실적 대안 마련을 동시에 달성.

## 5. 향후 계획 (Next Steps)
- 실제 상담 데이터 기반의 파인튜닝.
- 모바일 앱 또는 웹 UI에서의 대화 시각화(말풍선 디자인 등).
