# 🎓 인하공업전문대학 신입생 도우미 챗봇 UI 기획안

> **"사용자 경험(UX) 중심의 학사 안내 혁신"** > 본 문서는 인하공업전문대학 컴퓨터정보공학과 신입생들을 위한 스마트한 대학 생활 가이드의 핵심 UI/UX 디자인과 세부 시나리오를 정의합니다.

---

## 📌 1. UI/UX 기획 비전 (Design Vision)

* **친근함과 소속감 부여:** 마스코트 '인덕이'를 활용해 딱딱한 행정 정보를 친근하게 전환합니다.
* **접근성 극대화:** 복잡한 메뉴 탐색 없이, 단 1~2번의 클릭(Quick Buttons)으로 핵심 정보에 도달하도록 설계했습니다.
* **학과 맞춤형 가이드:** 컴퓨터정보공학과 신입생을 위한 전용 위치 정보를 제공하여 실용성을 높였습니다.

---

## 🎨 2. 디자인 시스템 & 마스코트

### 🎨 주요 테마 컬러 (Color Palette)
* **Inha Blue (`#004680`):** 학교의 공식 컬러로, 신뢰와 전문성을 상징합니다.
* **Inha Green (`#00843D`):** 서브 컬러로 사용되어 자연스럽고 편안한 느낌을 줍니다.
* **Light Gray (`#F1F5F9`):** 배경색 및 기본 말풍선 색으로 가독성을 높입니다.

## 📱 3. 사용자 여정 중심의 UI 시나리오

### [시나리오 01] 메인 화면 및 웰컴 인터랙션
* **UI 특징:** 화면 로딩 시 가장 먼저 웰컴 카드와 마스코트가 등장하여 소속감을 줍니다.
* **사용자 행동:** 텍스트 입력 없이 하단의 학사일정, 학생식당, 모바일 학생증 버튼을 눌러 빠른 답변을 유도합니다.

### [시나리오 03] 컴퓨터정보공학과 전용 가이드
* **UI 특징:** 일반적인 전체 공지를 넘어 특정 학과 학생을 위한 맞춤형 대화 분기를 생성합니다.
* **사용자 행동:** 1학년과 2, 3학년의 강의실 위치가 다르다는 점을 명확히 인지하고, 4호관 401호 학과 사무실로 헤매지 않고 찾아갈 수 있습니다.

## 🚀 4. 기대 효과 및 미래 가치

* **학사 안내 효율화:** 반복적인 단순 문의(학과 사무실 위치, 학생증 발급 방법 등)를 40% 이상 감소시켜 학과 조교 및 사무실의 업무 피로도를 낮춥니다.
* **신입생 만족도 상승:** 학교 정보 검색 시간을 단축시키고 대학 생활 적응을 돕는 필수 서비스로 자리 잡습니다.
* **확장성:** 추후 도서관 도서 검색 API, 셔틀버스 시간표 조회 등 추가 모듈 연동을 통해 종합 캠퍼스 플랫폼으로 확장 가능합니다.
