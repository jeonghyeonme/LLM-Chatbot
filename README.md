# LLM-Chatbot

LLM(Large Language Model)과 RAG(Retrieval-Augmented Generation) 기술을 활용한 지능형 챗봇 프로젝트입니다.

## 🚀 프로젝트 개요
- **목표**: 개인화된 지식 베이스를 바탕으로 정확하고 자연스러운 답변을 제공하는 챗봇 구현
- **주요 기술**: LLM (OpenAI, Anthropic, or Local LLMs), RAG (LangChain, LlamaIndex), Vector DB

## 📂 디렉토리 구조
- `/src`: 소스 코드 (Core logic, API, RAG implementation)
- `/docs`: 설계 문서 및 API 명세서
- `/data`: RAG 구현을 위한 지식 베이스 데이터셋
- `/tests`: 단위 테스트 및 통합 테스트 코드
- `/.github`: PR 및 이슈 템플릿
